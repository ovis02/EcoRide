# EcoRide

EcoRide est une plateforme web innovante pour la gestion et la réservation de véhicules, visant à réduire l'impact environnemental des déplacements en encourageant le covoiturage. 

## Technologies utilisées

Le projet **EcoRide** repose sur les technologies suivantes pour garantir une interface moderne et dynamique, ainsi qu'une gestion optimisée du backend :

- **HTML** : Pour la structure de base des pages web.
- **CSS** : Pour la mise en forme et le design des pages.
- **Bootstrap** : Framework CSS pour une conception responsive et rapide.
- **JavaScript** : Pour la gestion de l'interactivité côté client.
- **Symfony** : Framework PHP pour la création d'API et la gestion des données côté serveur.
- **PHP** : Pour le traitement des requêtes côté serveur et la gestion de la logique métier.
- **MySQL** et **MongoDB** : Bases de données pour stocker les informations relatives aux utilisateurs, aux véhicules et aux réservations.
